# 供应商绩效评分体系

## 一、评估维度及权重

| 维度 | 权重 | 评分标准 |
|------|------|---------|
| 质量 (Quality) | 35% | 来料合格率、客诉率、质量改进响应 |
| 交付 (Delivery) | 30% | 准时交付率、交付周期达标率 |
| 成本 (Cost) | 20% | 价格竞争力、成本优化配合度 |
| 服务 (Service) | 15% | 沟通响应速度、售后支持、配合度 |

## 二、评分等级

| 等级 | 分数范围 | 管理策略 |
|------|---------|---------|
| A（优秀） | 90-100 | 优先分配订单，考虑战略合作 |
| B（良好） | 75-89 | 正常合作，持续改进 |
| C（合格） | 60-74 | 限期整改，减少份额 |
| D（不合格） | <60 | 暂停合作，启动替代供应商 |

## 三、评估周期

- 月度：关键指标跟踪（质量、交付）
- 季度：综合评分发布
- 年度：供应商分级调整 + 淘汰/引入

## 四、关键指标计算公式

### 准时交付率
```
准时交付率 = 准时交付批次 / 总交付批次 × 100%
```

### 来料合格率
```
来料合格率 = 合格批次 / 总检验批次 × 100%
```

### 综合评分
```
综合评分 = 质量得分 × 0.35 + 交付得分 × 0.30 + 成本得分 × 0.20 + 服务得分 × 0.15
```
