# 入库/出库标准操作流程 (SOP)

## 一、入库流程

### 1. 到货签收
- 核对送货单与采购订单（PO）信息一致性
- 检查外包装完整性，异常拍照记录
- 签收后 2 小时内完成系统到货登记

### 2. 质检
- 按 AQL 标准抽检（一般检验水平 II）
- 检验结果录入 QMS 系统
- 不合格品隔离至"待退区"，24 小时内通知采购

### 3. 上架入库
- 根据 SKU 属性分配库位（ABC 分类法）
- A 类商品：拣货区黄金库位
- B 类商品：标准库位
- C 类商品：高层/远端库位
- 扫码确认入库，系统库存实时更新

## 二、出库流程

### 1. 订单下发
- OMS 推送拣货任务至 WMS
- 系统自动分配拣货路线（最短路径算法）

### 2. 拣货
- 手持终端引导拣货
- 逐件扫码核实 SKU 和数量
- 异常（缺货/破损）实时上报

### 3. 复核打包
- 核对订单明细与实物一致性
- 标准包装规范：泡沫填充 + 外箱封口
- 打印面单，粘贴至外箱右上角

### 4. 交接发运
- 按承运商分拣到对应发货区
- 逐箱扫码出库
- 交接单双方签字确认

## 三、库存预警规则

| 预警类型 | 触发条件 | 处理方式 |
|---------|---------|---------|
| 安全库存预警 | 可用库存 < 安全库存量 | 自动触发补货申请 |
| 滞销预警 | 90 天零销售 | 推送运营评估 |
| 效期预警 | 保质期剩余 < 总保质期 1/3 | 优先出库 + 促销处理 |
| 库容预警 | 库位使用率 > 85% | 通知仓储经理扩容 |
