# 退换货政策（2026 版）

## 一、退货条件

### 无理由退货
- 自签收之日起 7 天内
- 商品未经使用、不影响二次销售
- 原包装完整，配件齐全

### 质量问题退货
- 自签收之日起 15 天内
- 需提供质量问题照片/视频证据
- 由卖家承担退货运费

## 二、换货流程

1. 买家提交换货申请，附描述及照片
2. 客服审核（24 小时内回复）
3. 审核通过后发出退货地址
4. 收到退回商品质检合格后，48 小时内发出新品
5. 换货仅限同 SKU 更换，不支持换款

## 三、退款时效

| 退款场景 | 处理时限 |
|---------|---------|
| 仅退款（未发货） | 申请后 2 小时内处理 |
| 退货退款（已收到退回商品） | 质检通过后 3 个工作日内 |
| 质量问题退款 | 审核通过后 5 个工作日内 |
| 拒签退款 | 物流确认退回后 3 个工作日内 |

## 四、不支持退换货的情形

1. 定制类商品
2. 易耗品已拆封（如密封件、滤芯）
3. 人为损坏
4. 超出退换货时限
5. 无法提供有效购买凭证
